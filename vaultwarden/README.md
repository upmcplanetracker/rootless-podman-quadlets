# [Vaultwarden](https://github.com/dani-garcia/vaultwarden)

Self-hosted password manager.

## Access
Open `http://<your-ip>:8080` after starting.

## Volumes
Data is stored in `~/vaultwarden/data` by default. Change the left‑side path in the
`.container` file if you want a different location.

## Secrets / API keys
These have been left out. Please see the config information for Vaultwarden for more information.
